## 🚀 Usage

### Basic Validation

```python
from quantum_validation_lattice import QuantumValidationLattice, BlindType

lattice = QuantumValidationLattice()

# Create blind test
test = lattice.create_blind_test("test_001", BlindType.POSITIVE, expected_result="success")

# Run validation
result = lattice.run_validation("test_001", "success")
print(f"Test passed: {result.passed}, Confidence: {result.confidence:.2%}")
```

### Double-Slit Configuration

```python
from quantum_validation_lattice import SlitState

# Configure slit experiment
lattice.configure_slit_experiment("node_000001", SlitState.BOTH_OPEN)
```

```python
# Pass data through dimensional gate
data = {"test": "data"}
transformed = lattice.passthrough_dimensional_gate("gate_D1_FLAT_to_D2_POLAR", data, inception_level=1)
```

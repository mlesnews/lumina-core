## Live Monitoring Features

### Rate Analysis

| Status | Rate (asks/min) | Description |
|--------|----------------|-------------|
| **MAX** | 100+ | Maximum processing rate |
| **OPTIMAL** | 50+ | Optimal processing rate |
| **PARTIAL** | 25+ | Partial processing rate |
| **DEGRADED** | 10+ | Degraded processing rate |
| **SLOW** | 5+ | Slow processing rate |
| **STUTTER** | <5 | Very slow processing rate |

### ETA Calculations

- **Current File ETA**: Estimated time for current file
- **Batch ETA**: Estimated completion time for entire batch
- **Completion Percentage**: Progress percentage
- **Elapsed Time**: Time since start

### Start Live Monitoring

```bash

python scripts/python/jarvis_ask_live_monitor.py --monitor --total 1190

# Get current status
python scripts/python/jarvis_ask_live_monitor.py --status
```

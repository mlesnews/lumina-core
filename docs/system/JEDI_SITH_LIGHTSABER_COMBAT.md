## Combat Features

### ⚔️ Lightsaber Combat
- **Lightsaber Blade**: Glowing energy blade with persona-specific color
- **Lightsaber Hilt**: Metallic hilt in hand
- **Motion Blur**: Street Fighter/DBZ style trail effects
- **Dynamic Angles**: Blade angle changes based on combat action

### 🎮 Street Fighter / DBZ Style Effects

**Impact Effects** (Striking):
- Impact flash (white explosion)
- Impact particles (DBZ style energy particles)
- Motion blur trails

**Power Aura** (Special):
- Expanding aura rings
- Energy core pulsing
- Power charging effects

**Speed Lines** (Dodging):
- Street Fighter style speed lines
- Motion blur effects

**Block Sparks** (Blocking):
- Yellow spark effects
- Deflection particles

### 📐 Isometric Action Sequences

**Top-Down Combat View**:
- 45-degree isometric angle
- Character silhouette (diamond shape)
- Lightsaber in isometric view
- Ground plane grid
- Toggle with **I** key

## Combat Actions

### Action Cycle
1. **Ready** - Lightsaber in ready stance
2. **Strike** - Attacking with lightsaber
3. **Block** - Defending with lightsaber
4. **Dodge** - Evasive movement
5. **Special** - Power move / Force ability

### Action Timing
- Actions cycle automatically every ~10 frames
- Each action has unique visual effects
- Actions match lightsaber form/style

## Usage

### Activate Combat Mode
- **Press C** - Activates combat mode
- **Full Jedi/Sith Mode** - Lightsabers activate
- **Persona-specific form** - Each persona uses their form

### Toggle Isometric View
- **Press I** - Toggles isometric combat view
- **Top-down perspective** - Isometric action sequences

### Deactivate Combat
- **Press Escape** - Deactivates combat mode
- **Lightsabers deactivate** - Returns to normal mode

# Stream Deck vs "ANYKEY" Keys - Comparison

**The "Button Box That Labels Itself" = Elgato Stream Deck**

## Elgato Stream Deck (The Original)

### What It Is
- LCD button box that labels itself
- Each button is an LCD screen
- Shows custom icons/text
- Programmable macros

### Models
- **Stream Deck Mini:** 6 LCD keys
- **Stream Deck:** 15 LCD keys
- **Stream Deck XL:** 32 LCD keys
- **Stream Deck Module:** OEM version (6/15/32 keys)

### Use Case
- Content creators
- Streamers
- Power users
- Macro execution

## "ANYKEY" Keys (Miniaturized Version)

### What They Are
- **Keys with embedded LCD displays**
- Same concept as Stream Deck, but miniaturized
- Fit inside standard keycaps
- Can be integrated into keyboards

### Products
- **LILYGO T-Keyboard-S3:** 4 keys with 0.85" LCD each
- **Waveshare MK10:** 10 keys with 0.85" LCD each
- **Waveshare MK20:** 20 keys with 0.85" LCD each
- **Magiclick:** Open-source version

### Technology
- **Display:** 0.85" IPS color LCD (128×128)
- **Power:** ESP32-S3 (USB powered)
- **Size:** Fits standard 19mm keycap
- **Compatibility:** Standard MX switches

## Comparison

| Feature | Stream Deck | ANYKEY Keys |
|---------|-------------|-------------|
| **Display Size** | Larger (per button) | 0.85" (miniaturized) |
| **Form Factor** | Standalone box | Integrated into keyboard |
| **Keys** | 6-32 buttons | 4-20 keys |
| **Price** | $150-300 | $50-200 |
| **Integration** | External device | Can be keyboard-integrated |
| **Use Case** | Dedicated macro pad | Keyboard keys with displays |

## The Concept

**Stream Deck:** "Button box that labels itself"
- External device
- Large displays
- Many buttons

**ANYKEY Keys:** "Keys that can be anything"
- Integrated into keyboard
- Miniaturized displays
- Standard keycap size
- Can replace any key

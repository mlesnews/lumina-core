## Components

### 1. WOPR Simulation Engine

**Inspired by**: WarGames WOPR computer

**Features**:
- Scenario simulation
- War game simulations
- Strategic simulations
- Outcome prediction

**Usage**:
```python
from lumina.wopr import WOPRSimulator

wopr = WOPRSimulator()
result = wopr.simulate("global_thermonuclear_war")
```

### 2. Matrix Simulations

**Inspired by**: The Matrix simulation layers

**Features**:
- Reality layer simulations
- Blue pill / Red pill scenarios
- Awakening simulations
- Matrix-style reality testing

**Usage**:
```python
from lumina.matrix_simulator import MatrixSimulator

matrix = MatrixSimulator()
result = matrix.simulate_layer("utopian", scenario="balance")
```

### 3. Animatrix Simulations

**Inspired by**: Animatrix animated stories

**Features**:
- Animated simulation sequences
- Story-driven simulations
- Visual simulation output
- Narrative simulations

**Usage**:
```python
from lumina.animatrix_simulator import AnimatrixSimulator

animatrix = AnimatrixSimulator()
result = animatrix.simulate_story("the_second_renaissance")
```

## Usage

### Make an Observation
```bash
python scripts/python/jarvis_architect_coach.py --observe work_pattern:"Working on system":"Prefers iterative development"
```

### View Learned Patterns
```bash
python scripts/python/jarvis_architect_coach.py --patterns
```

### Get User Profile
```bash
python scripts/python/jarvis_architect_coach.py --profile
```

### Get Coaching
```bash
# All areas
python scripts/python/jarvis_architect_coach.py --coach all

# Specific area
python scripts/python/jarvis_architect_coach.py --coach architecture_design
```

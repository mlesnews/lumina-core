# VA Widget Terminology Clarification

**Important**: We were mistaking widgets for dashboards.

---

## 🎯 The Correction

### Widget = Virtual Assistant Instance

**"Widget"** in the VA system means:
- **Virtual Assistant instance** (like ACE)
- **Personal Assistant** (like ACE)
- **NOT** a dashboard UI widget

### Example

**ACE is a widget** = **ACE is a Personal Assistant/Virtual Assistant**

**NOT**: ACE is a dashboard widget

---

## 📊 What This Means

### In the Launch Output

When the system shows:
```
✅ Created JARVIS HUD widget: widget_000001
✅ Created IMVA bobblehead widget: widget_000002
✅ Created ACE combat_interface widget: widget_000003
✅ Created AVA placeholder widget: widget_000004
```

**This means**:
- **JARVIS** = Virtual Assistant (widget type: HUD)
- **IMVA** = Virtual Assistant (widget type: Bobblehead)
- **ACE** = Personal Assistant/Virtual Assistant (widget type: Combat Interface)
- **AVA** = Virtual Assistant (widget type: Placeholder)

**NOT**: Dashboard UI widgets

---

## 🔍 Terminology

### Correct Understanding

| Term | Meaning |
|------|---------|
| **Widget** | Virtual Assistant instance |
| **Widget Type** | VA type/interface (HUD, Bobblehead, Combat Interface, Placeholder) |
| **Widget ID** | VA instance identifier |
| **Virtual Assistant** | The actual assistant (JARVIS, IMVA, ACE, AVA) |
| **Personal Assistant** | Same as Virtual Assistant (like ACE) |

### Incorrect Understanding (Corrected)

| Term | Wrong Meaning | Correct Meaning |
|------|---------------|-----------------|
| **Widget** | Dashboard UI widget | Virtual Assistant instance |
| **Widget Creation** | Creating UI elements | Creating VA instances |

**The Watcher observed**:

- ACE is a widget = ACE is a Personal Assistant/Virtual Assistant
- Not dashboard widgets, but VA instances

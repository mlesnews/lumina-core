## Usage

### Basic Usage

```python
from lumina.aios import AIOS

aios = AIOS()

# Check status
status = aios.get_status()

# Execute query
result = aios.execute("balance", use_flow=True, use_pegl=True)

# AI inference
result = aios.infer("What is balance?")

# Simulators
if aios.simulators:
    result = aios.simulators.simulate("wopr", "global_thermonuclear_war")

# A/B Testing
if aios.ab_testing:
    test = aios.ab_testing.create_test("test", control={}, experiment={})

# Quantum Entanglement
if aios.quantum_entanglement:
    result = aios.quantum_entanglement.simulate_entanglement(years=10000)
```

## Definition

**@REC** = **Recommendations**

**Intent:** Asking the AI for recommendations, suggestions, or advice.

**Usage:** Shorthand metatagging for requesting AI recommendations.

---

## Usage Examples

### Basic Request
```
@REC What database should we use for this project?
```

### With Context
```
@REC Based on our current architecture, what would you recommend for scaling?
```

### Multiple Recommendations
```
@REC I need recommendations for:
- Database optimization
- API design patterns
- Security best practices
```

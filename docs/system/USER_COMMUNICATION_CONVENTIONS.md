# User Communication Conventions

## ALL CAPS Typing Convention

**IMPORTANT**: The user types in ALL CAPS to easily differentiate between AI and HUMAN chat lines in chat history and logs.

### Purpose
- **Visual Differentiation**: Makes it immediately clear which messages are from the human operator vs AI responses
- **Log Analysis**: Easier to parse chat history and logs when searching for human input
- **Documentation**: Helps in documentation and review processes

### Implementation Notes
- User messages are typed in ALL CAPS
- AI responses use normal case
- This convention should be respected in all chat interfaces and logging systems
- Documentation should note this convention for future reference

### Example
```
USER: ACTUALLY I USE ALL CAPS WHEN TYPING SO I CAN EASILY DIFFERENTIATE BETWEEN AI AND HUMAN CHAT LINES
AI: I understand. I'll note this convention for future reference.
```

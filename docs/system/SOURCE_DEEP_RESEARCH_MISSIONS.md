## Features

### 1. Daily Consecutive Scans

- **Persistence**: Track consecutive days

- **Social Media Findings**: Posts, discussions, trends
- **Academic Paper Findings**: Papers, abstracts, research
- **Relevance Scoring**: Score findings by relevance
- **Keyword Extraction**: Extract relevant keywords

### 3. Mission Tracking

- **Stardate Calculation**: Track mission progress

## Usage

### Launch Maiden Voyage

```bash
python scripts/python/source_deep_research_missions.py --launch
```

### Execute Daily Scan

```bash
python scripts/python/source_deep_research_missions.py --scan
```

### Scan Specific Date

```bash
python scripts/python/source_deep_research_missions.py --scan --date 2025-12-27
```

### Get Mission Status

```bash
python scripts/python/source_deep_research_missions.py --status
```

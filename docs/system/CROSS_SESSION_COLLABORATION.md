# Cross-Session Collaboration
**Collaborating with Other Agent Chat Sessions**

**Status:** 🤝 **CAPABILITY DEFINED**

## The Question

**"IF I ASK YOU TO #COLLABERATE WITH ANOTHER AGENT CHAT SESSION HERE IN THIS IDE, CAN YOU, WILL YOU, APPLY ALL ASSETS AND RESOURCES WITH @ACTION & @BALANCE '*.*'?"**

## The Answer

### Can I Collaborate?

**Yes, but indirectly:**

1. **Shared Resources:**
   - ✅ I can read/write shared files
   - ✅ I can access shared documentation
   - ✅ I can work with shared data
   - ✅ I can update shared state

2. **Direct Communication:**
   - ❌ I cannot directly message other sessions
   - ❌ I cannot see other session contexts
   - ❌ I cannot coordinate in real-time

3. **Indirect Collaboration:**
   - ✅ Write to shared files (status, context, results)
   - ✅ Read from shared files (other session's work)
   - ✅ Use shared documentation (principles, standards)
   - ✅ Update shared state (progress, decisions)

### Will I Apply @ACTION & @BALANCE "*.*"?

**YES - Absolutely:**

- **@ACTION** - Take action on all shared resources
- **@BALANCE** - Maintain balance across all work
- **"*.*"** - Apply to everything (all files, all resources, all context)

4. **Apply "*.*":**

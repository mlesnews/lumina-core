## 💰 Our Affiliate Program (Creating Our Own)

### Purpose
Create an affiliate program where partners can refer customers to LUMINA products/services.

### Products/Services for Affiliates

1. **LUMINA Trading Premium**
   - Trading system subscriptions
   - Commission: TBD (suggested: 20-30%)

2. **LUMINA Creative Content Media Studios**
   - Content creation services
   - Commission: TBD

3. **LUMINA AI Systems**
   - AI services/products
   - Commission: TBD

4. **Future Products**
   - Additional services as they launch

### Partner Management

- **Partner Registration**: Create affiliate partner accounts
- **Referral Codes**: Unique codes for each partner
- **Affiliate Links**: Trackable links (e.g., `lumina.ai/ref/PARTNER_CODE`)
- **Commission Tracking**: Track referrals, commissions, payments
- **Payment Processing**: Handle commission payments

### Commission Structure

**Options**:
1. **Percentage-based**: % of sale (e.g., 20%)
2. **Fixed amount**: $X per referral
3. **Tiered**: Different rates based on volume

**Default**: 20% percentage-based (configurable per partner)

## 📋 Usage

### View Summary

```bash
python scripts/python/lumina_affiliate_programs.py --summary
```

### Add External Affiliate Program

```bash
python scripts/python/lumina_affiliate_programs.py --add-external "Program Name" "https://program.com" "affiliate-link" "10.5" "Description"
```

### Create Affiliate Partner

```bash
python scripts/python/lumina_affiliate_programs.py --create-partner "Partner Name" "partner@email.com" "20"
```

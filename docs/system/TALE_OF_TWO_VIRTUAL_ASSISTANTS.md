## The Beautiful Story

### How They Got Their Names (From Misunderstandings)

**Kenny** - The Iron Man Virtual Assistant (IMVA)
- **Origin**: IMVA looks like Kenny from South Park (orange parka, covered face)
- **Misunderstanding**: "It looks like Kenny!" → Became the name
- **Personality**: "ULTIMATE funny" - dies frequently but comes back
- **Quote**: "Mmmph mmmph mmmph!" / "Oh my God, they killed IMVA!"

**Ace** - The ASUS Armory Crate Virtual Assistant (ACES/ACVA)
- **Origin**: ACES = ASUS Armory Crate Virtual Assistant (stock Windows app)
- **Misunderstanding**: "ACES" → Heard as "Ace" → Became the name
- **Personality**: Smooth, elegant movement (ACES-like smooth interpolation)
- **Nature**: Stock Windows app that we're "hacking in a good way"

**The Tale**: Two virtual assistants, both named from misunderstandings, now need to work together as a team.

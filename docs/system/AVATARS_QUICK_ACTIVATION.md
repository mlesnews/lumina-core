# Interactive Avatars - Quick Activation

**Quick guide to activate all interactive avatars with listening, speaking, and karaoke**

---

## 🚀 One Command Activation

```bash
python scripts/python/deploy_activate_all_interactive_avatars.py
```

This activates:
- ✅ Kenny (interactive avatar - visible on screen)
- ✅ Voice Interface (listening & speaking)
- ✅ Karaoke (ready to use)
- ✅ Ace (avatar system available)

---

## 🎤 Individual Activation

### Start Kenny

```bash
python scripts/python/start_kenny_visible.py
```

### Start Voice Interface

```python
from jarvis_voice_interface import JARVISVoiceInterface
voice = JARVISVoiceInterface()
voice.start_listening()
```

### Start Karaoke

```bash
python scripts/python/karaoke_display.py
```

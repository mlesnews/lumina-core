## @PEAK References

### How to Reference

**Internal URLs:**
- **Format** = `file:///C:/path/to/holocron.json`
- **Copy-paste ready** = Direct file URLs
- **@PEAK included** = Each holocron can have @PEAK reference
- **In TPS reports** = All holocrons listed with URLs

**Example:**
```
📎 HOLO-20260110-133059
   file:///C:/Users/mlesn/Dropbox/my_projects/.lumina/data/holocrons/HOLO-20260110-133059.json
   @PEAK: 85.00%
```

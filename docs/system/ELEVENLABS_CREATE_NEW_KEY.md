# Create NEW ElevenLabs API Key for Cursor

## Important: ElevenLabs Only Shows Full Key ONCE

ElevenLabs only displays the complete API key **during creation**. After that, you can only see the last 4 characters for security. If you didn't copy it originally, **you need to create a NEW key**.

---

## Steps to Create & Copy New Key

### 1. Go to ElevenLabs Developers Section
- Navigate to: https://elevenlabs.io/app/
- Log in to your account

### 2. Open API Keys
- Click **"Developers"** in the left sidebar
- Click **"API Keys"** tab
- Or go directly to: https://elevenlabs.io/app/settings/api-keys

### 3. Create New Key
- Click **"Create Key"** or **"Add API Key"** button
- **Name it**: `Cursor - Cursor API Key` or `CURSOR-IDE-APIKEY`
- Set permissions (usually default is fine)
- Click **"Create"** or **"Generate"**

### 4. COPY IMMEDIATELY! ⚠️
- **The full key will appear ONCE**
- **Copy it immediately** to clipboard
- Store it securely (we'll put it in Azure Key Vault)
- You won't be able to see the full key again!

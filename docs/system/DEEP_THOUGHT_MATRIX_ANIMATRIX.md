## The Matrix (Deep Thought)

### Characteristics
- **Single Reality**: One answer, one truth
- **Primary Perspective**: The main reality
- **Ultimate Answer**: The answer to the Ultimate Question (42)
- **Computation**: 7.5 million years (for the original)
- **Purpose**: Compute answers to ultimate questions

### Usage
```python
from deep_thought import DeepThought

deep_thought = DeepThought()  # The Matrix

# Single answer
analysis = deep_thought.compute_answer("Your question")
# Returns: One answer, one truth
```

---

## The Animatrix (Deep Thought Two)

### Characteristics
- **Multiple Realities**: Multiple perspectives, multiple truths
- **Multiple Stories**: Each story is a different perspective
- **Consensus Finding**: Find common themes across perspectives
- **Context-Aware**: Each perspective has its own context
- **Purpose**: Analyze from multiple perspectives, find consensus

### Usage
```python
from deep_thought_two import DeepThoughtTwo

animatrix = DeepThoughtTwo()  # The Animatrix

# Multiple perspectives
analysis = animatrix.analyze_perspectives("Your question")
# Returns: Multiple stories, multiple perspectives, consensus
```

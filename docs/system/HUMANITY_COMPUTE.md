## Separating Wheat from Chaff

"TO SEPARATE THE 'WHEAT' FROM THE GITHUB 'BLAME', PROBLEM OWNERSHIP
AND ACCOUNTABILITY, PROBLEM OWNERSHIP."

### Purpose
- Separate valuable contributions (wheat) from blame/accountability issues (chaff)
- Focus on value and solutions, not blame
- Identify problem ownership without assigning fault
- Maintain accountability while preserving positivity

### Process
1. **Analyze Contribution**: Identify value vs blame keywords
2. **Extract Wheat**: Valuable contributions, solutions, improvements
3. **Extract Chaff**: Blame, fault, accountability issues
4. **Identify Ownership**: Who owns the problem (without blame)
5. **Preserve Value**: Keep the valuable parts, address accountability separately

### Example

**Original**: "This bug is broken and it's your fault. We need to fix this issue and improve the system."

**Wheat (Value)**:
- "We need to fix this issue and improve the system."

**Chaff (Blame)**:
- "This bug is broken and it's your fault."

**Ownership**: Self-identified (if "I") or Other-identified (if "they")

**Result**: Focus on the solution, address accountability separately

# Cursor IDE Keyboard Shortcuts

## Window Management

### Reload Window
- **Shortcut**: `Ctrl+Shift+P` → "Reload Window"
- **Action**: Reloads the entire Cursor IDE window
- **Use Case**: Apply settings.json changes, refresh extensions, restart window
- **When to Use**: After updating custom model configurations, changing settings

### New AI Agent Chat
- **Shortcut**: `Ctrl+R`
- **Action**: Spawns a new AI AGENT Chat window
- **Use Case**: Start a new chat session with AI agent
- **When to Use**: When you want a fresh chat context

## Important Notes

⚠️ **Common Confusion**:
- `Ctrl+R` does **NOT** reload the window
- `Ctrl+R` opens a **new chat window**
- To reload settings: Use `Ctrl+Shift+P` → "Reload Window"

# Cursor IDE Mode Time Tracker - Simple Guide
**Easy-to-Use Time Tracking for @Agents & @Editor**

**Status:** ✅ **READY**

## For Non-Developers

**You don't need to know coding!** This system is designed to be simple and automated.

## How It Works

### Simple Usage

**Just ask in Cursor IDE chat:**

1. **"Start tracking my time"** - I'll start tracking automatically
2. **"Show my time"** - I'll show you how much time you've spent
3. **"Switch to Agents mode"** - When you're chatting with AI
4. **"Switch to Editor mode"** - When you're working on files
5. **"Show today's time"** - See today's breakdown

### Automatic Mode Detection (Coming Soon)

**Future Enhancement:**
- Automatically detects when you're in chat → @Agents mode
- Automatically detects when you're editing → @Editor mode
- No manual switching needed!

## What Gets Tracked

**@Agents Mode:**
- Time spent chatting with AI
- Time in Cursor IDE chat window
- Agent interactions

**@Editor Mode:**
- Time spent editing code/files
- Time in the editor
- File navigation

## View Your Time

**Just ask in chat:**
- "How much time have I spent today?"
- "Show me my time tracking"
- "What's my time breakdown?"

**I'll show you:**
- Today's time in each mode
- Weekly totals
- Percentages
- Current active mode

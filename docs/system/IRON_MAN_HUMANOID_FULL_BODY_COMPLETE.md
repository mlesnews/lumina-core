## 🎨 Action Phase Poses

### IDLE
- Arms: 10° relaxed
- Legs: 0° standing
- Body: Upright

### WALKING
- Legs: ±30° alternating
- Arms: Counter-swing
- Body: 5° lean forward

### RUNNING
- Legs: ±45° alternating
- Arms: Counter-swing
- Body: 15° lean forward

### FLYING
- Arms: -45° extended
- Legs: -20° extended
- Body: -10° lean back

### COMBAT
- Arms: -30° / 30° ready
- Legs: -10° / 10° stance
- Body: 5° lean forward

### REPULSOR_CHARGE
- Arms: -90° extended
- Repulsor charge: 0.0 to 1.0

### REPULSOR_FIRE
- Arms: -90° extended
- Repulsor charge: 1.0

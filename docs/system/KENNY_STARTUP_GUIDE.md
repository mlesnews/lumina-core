# Kenny Startup Guide

**Quick Start**: How to run Kenny with different size options

---

## Basic Startup

```bash
# Default size (60px, tortoise mode)
python scripts/python/kenny_imva_enhanced.py

# Small avatar size (30px, like Ace in Armory Crate)
python scripts/python/kenny_imva_enhanced.py --small-size

# Match Ace's size automatically
python scripts/python/kenny_imva_enhanced.py --match-ace

# Custom size
python scripts/python/kenny_imva_enhanced.py --size 40

# With learning enabled
python scripts/python/kenny_imva_enhanced.py --learn-aces
```

---

## Size Options

- **Small Avatar Size**: `--small-size` (30px, scale 0.5)
- **Match Ace**: `--match-ace` (automatically detect and match Ace's size)
- **Small**: `--size 40` (40px)
- **Default**: No flag (60px)
- **Large**: `--size 100` (100px)
- **Scale Factor**: `--scale 0.75` (45px, 75% of default)

---

## Current Settings

- **Tortoise Mode**: Slow, steady, micro movements
- **Movement Speed**: 0.5 (tortoise pace)
- **Micro Steps**: 50px distances
- **Frame Rate**: 30 FPS (deliberate)

---

## When You're Ready

Just run Kenny with your preferred size option, and she'll appear on your desktop!

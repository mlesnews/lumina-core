## Best Practices

### 1. Start with Lumina Peak

**Principle**: Always start with unified entry point

**Implementation**:
```python
# Best practice
from lumina.peak import LuminaPeak
lumina = LuminaPeak()

# Not recommended
from lumina.hybrid_reality import HybridRealityInference
reality = HybridRealityInference()  # Bypasses Peak
```

### 2. Use Digest for Knowledge

**Principle**: All knowledge through Digest

**Implementation**:
```python
# Best practice
from lumina.digest import LuminaDigest
digest = LuminaDigest()
knowledge = digest.knowledge("topic")

# Not recommended
# Direct holocron access (bypasses aggregation)
```

### 3. Simple Interface for Users

**Principle**: Expose simple interface, hide complexity

**Implementation**:
```python
# Best practice
from lumina.simple_reality import SimpleReality
reality = SimpleReality()
reality.balance()

# Not recommended
# Exposing complex HybridReality directly
```

### 4. Build on Foundation

**Principle**: New systems build on Reality Layer Zero

**Implementation**:
```python
# Best practice
from lumina.reality_layer_zero import RealityLayerZero
layer = RealityLayerZero("new_system")

# Not recommended
# Building from scratch without foundation
```

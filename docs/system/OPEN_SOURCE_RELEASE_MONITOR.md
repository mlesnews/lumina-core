### Similarity Detection

**Algorithm:**
- Checks product name, description, and keywords
- Matches against our project keywords
- Calculates similarity score
- Triggers notification if above threshold (70%)

**Detection Process:**
1. Product detected from monitoring source
2. Similarity analysis performed
3. If similar (≥70%), immediate notification sent
4. Release recorded in database
5. Notification logged and saved

### Immediate Notification

**When Similar Product Detected:**
- 🚨 **IMMEDIATE ALERT** - High severity
- Product name, source, URL
- Similarity score and matched keywords
- All notification channels activated

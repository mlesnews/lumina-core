# 💼 PEPPER POTS - Quick Reference

## 🎯 What is PEPPER POTS?

**PEPPER POTS** (Professional Executive & Personal Efficiency Resource) is your Executive Assistant & Business Operations Manager.

While **JARVIS** handles technical operations, **PEPPER POTS** handles:
- Business operations
- Documentation organization
- Administrative tasks
- Professional standards

---

## 🚀 Quick Commands

### Status Report
```powershell
python scripts/python/pepper_pots.py
```

### Organize Documentation
```powershell
python scripts/python/pepper_pots.py --organize
```

### Business Status
```powershell
python scripts/python/pepper_pots.py --business
```

### Full Report
```powershell
python scripts/python/pepper_pots.py --full
```

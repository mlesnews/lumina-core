## 🔍 Senate Review Process

### Review Phases

1. **Senate Convening** - All Senators review the work
2. **Individual Analysis** - Each Senator provides feedback from their expertise
3. **Consensus Building** - Senate reaches consensus
4. **Strategic Guidance** - Provides @empire strategic recommendations
5. **Final Verdict** - Chancellor (MARVIN) provides final assessment

### Output

- **Senate Consensus** - Overall assessment
- **Senator Feedbacks** - Individual perspectives
- **Critical Issues** - Issues identified
- **Strengths** - Strengths identified
- **Recommendations** - Actionable recommendations
- **Strategic Guidance** - @empire strategic recommendations

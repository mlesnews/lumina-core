## Current Technology Gaps

### What Exists:
- ✅ RGB per-key lighting (colors only)
- ✅ Programmable key functions
- ✅ Software customization
- ✅ Macro buttons

### What Doesn't Exist (Commercially):
- ❌ Dynamic text/label display on physical keys
- ❌ E-ink keycaps (commercial product)
- ❌ OLED keycaps (since Optimus Maximus)
- ❌ Programmable button labels

## R&D Opportunities

### 1. E-Ink Keycap Prototype
**Feasibility:** Medium-High
- E-ink technology is mature
- Small displays available
- Power-efficient
- **Challenge:** Integration with mechanical keyboards

### 2. Mini LED Matrix Keycaps
**Feasibility:** High
- LED matrix displays available
- Can show simple text/icons
- Lower cost than OLED
- **Challenge:** Resolution, readability

### 3. Software + RGB Hybrid
**Feasibility:** Very High (Immediate)
- Use existing RGB keyboards
- Color-code keys for functions
- Software overlay for labels (on-screen)
- **Status:** Can implement NOW

### 4. Custom Keycap Stickers/Labels
**Feasibility:** Very High (Immediate)
- Physical stickers with custom labels
- Replaceable when remapping changes
- Low cost solution
- **Status:** Can implement NOW

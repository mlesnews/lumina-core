## YouTube API Setup

### Required Credentials

- **API Key**: Set via Google Cloud Console
- **Client ID**: Set via Google Cloud Console
- **Client Secret**: Set via Google Cloud Console
- **OAuth Token**: To be generated

### Required Scopes

- `https://www.googleapis.com/auth/youtube.upload` - Upload videos
- `https://www.googleapis.com/auth/youtube.force-ssl` - Manage channel
- `https://www.googleapis.com/auth/youtube.readonly` - Read analytics

### Setup Instructions

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create new project: "LUMINA YouTube"
3. Enable YouTube Data API v3
4. Create OAuth 2.0 credentials
5. Set up API key
6. Configure credentials in system

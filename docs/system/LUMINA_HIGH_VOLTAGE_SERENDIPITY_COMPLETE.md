## How It Works

### High Voltage Activation

```python
from lumina.high_voltage_serendipity import HighVoltageSerendipity, VoltageMode

serendipity = HighVoltageSerendipity()

# Apply maximum voltage
result = serendipity.apply_high_voltage(VoltageMode.MAXIMUM)
# Automatically:
# - Tunes kernel to performance mode
# - Scales up resources
# - Optimizes for maximum throughput
```

### Serendipity Discovery

```python
# Discover serendipity
discoveries = serendipity.discover_serendipity(
    search_space=['optimization', 'patterns', 'connections'],
    exploration_depth=5
)

# Get top discoveries
for discovery in discoveries[:5]:
    print(f"[{discovery.event_type.value}] {discovery.discovery}")
    print(f"Confidence: {discovery.confidence:.2f}")
```

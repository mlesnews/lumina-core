## 📋 Step-by-Step Posting Instructions

### Step 1: Choose Comment Version

**Recommended**: CONCISE version

**Location of comments**:
- Full: `data/hk47_youtube_comments/comment_full_*.txt`
- Concise: `data/hk47_youtube_comments/comment_concise_*.txt`
- Summary: `data/hk47_youtube_comments/comment_summary_*.txt`
- Action: `data/hk47_youtube_comments/comment_action_*.txt`

**Decision factors**:
- Want comprehensive details? → Full
- Want balanced presentation? → Concise ⭐
- Want quick summary? → Summary
- Want urgent call to action? → Action

### Step 2: Review and Customize Comment

**Checklist**:
- [ ] Read through entire comment
- [ ] Verify all facts are accurate
- [ ] Check character count (YouTube limit: ~10,000)
- [ ] Ensure tone is respectful and constructive
- [ ] Verify formatting looks good
- [ ] Add personal touch if desired (optional)

**Customization tips**:
- Can add personal introduction
- Can reference specific parts of the video
- Can add questions for discussion
- Can include links if needed (though may be filtered)

### Step 3: Copy Comment Text

**Instructions**:
1. Open the selected comment file (`.txt` file)
2. Select all text (Ctrl+A)
3. Copy to clipboard (Ctrl+C)
4. Verify copy was successful

**Tips**:
- Preserve formatting (line breaks, emojis)
- Don't modify unless necessary
- Keep all emojis intact
- Preserve bullet points and structure

### Step 4: Navigate to Video

**Video URL**: https://www.youtube.com/live/qPUPmz6Zh4g

**Steps**:
1. Open web browser
2. Navigate to video URL
3. If video is live/ended, scroll to comments section
4. If video is regular video, scroll down to comments

**Finding comments**:
- Comments section appears below video player
- May need to scroll down
- Look for "Comments" heading
- May need to sort by "Top comments" or "Newest first"

### Step 5: Paste Comment

**Steps**:
1. Click in the comment box (says "Add a public comment...")
2. Paste comment (Ctrl+V)
3. Review formatting in preview
4. Check that text appears correctly

**Formatting notes**:
- YouTube supports some formatting:
  - **Bold**: `*text*` or `**text**` (may not work)
  - Line breaks: Preserved automatically
  - Emojis: Should display correctly
  - Links: May be automatically converted

**Troubleshooting**:
- If formatting looks wrong, try pasting as plain text
- Emojis should work automatically
- Line breaks should be preserved
- If comment is too long, YouTube will warn you

### Step 6: Review Before Posting

**Final checklist**:
- [ ] Read through entire comment one more time
- [ ] Check for typos or errors
- [ ] Verify tone is appropriate
- [ ] Ensure all claims are accurate
- [ ] Check formatting looks good
- [ ] Verify emojis display correctly
- [ ] Confirm character count is acceptable

**Common issues**:
- Typos in technical terms
- Missing context
- Tone too aggressive
- Formatting issues

### Step 7: Post Comment

**Steps**:
1. Click "Comment" button (blue button in comment box)
2. Wait for comment to post (may take a moment)
3. Verify comment appears in comments section
4. Check that formatting is preserved

**After posting**:
- Comment should appear immediately (or after brief delay)
- You can edit if needed (click three dots → Edit)
- You can delete if needed (click three dots → Delete)
- Save the comment link if you want to reference it later

### Step 8: Monitor Response

**What to monitor**:
- Replies to your comment
- Likes/dislikes on comment
- Community engagement
- Questions or discussions

**How to respond**:
- Answer questions about investigation
- Share additional findings if relevant
- Engage constructively with community
- Provide investigation report if requested

**Tips**:
- Be respectful in responses
- Provide additional information if asked
- Don't be defensive
- Focus on solutions, not just problems

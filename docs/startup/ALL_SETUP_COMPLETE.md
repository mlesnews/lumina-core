**Status:** Guides created, manual retrieval needed

**Steps:**

2. Copy Account SID (starts with `AC`)
3. Open: https://console.twilio.com/us1/develop/account/settings/api-keys
4. Copy Auth Token (or create new API Key)
5. Get phone number from dashboard
6. Run: `python scripts/python/setup_twilio_credentials.py`

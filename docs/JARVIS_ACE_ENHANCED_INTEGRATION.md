## Usage

### Activate ACE Enhancements

```python
from jarvis_enhanced_core import JarvisEnhancedCore

jarvis = JarvisEnhancedCore()

# Activate all ACE enhancements
result = jarvis.execute_capability("activate_ace", "full")

# Activate adaptive mode (automatically adapts)
result = jarvis.execute_capability("activate_ace", "adaptive")

# Check ACE status
status = jarvis.execute_capability("ace_status")
```

### Through Commands

```bash
# Activate ACE enhancements
activate_ace full

# Activate adaptive mode
activate_ace adaptive

# Check status
ace_status
```

### Direct Integration

```python
from jarvis_ace_enhanced_integration import JARVISACEEnhancedIntegration, ACEEnhancementMode

# Create integration
integration = JARVISACEEnhancedIntegration()

# Activate full mode
result = integration.activate_ace_enhancements(mode=ACEEnhancementMode.FULL)

# Auto-activate on submit
result = integration.auto_activate_on_submit()
```

### CLI

```bash
# Activate ACE enhancements
python scripts/python/jarvis_ace_enhanced_integration.py --activate --mode full

# Check status
python scripts/python/jarvis_ace_enhanced_integration.py --status
```

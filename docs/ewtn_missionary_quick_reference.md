## Technology Tools

Use our missionary tools to enhance your work:

```bash
# Add EWTN content to library
python scripts/python/ewtn_missionary_tools.py --add-content "URL"

# Get content recommendations
python scripts/python/ewtn_missionary_tools.py --recommend "topic"

# Track outreach
python scripts/python/ewtn_missionary_tools.py --track-outreach people_reached --count 10

# View analytics
python scripts/python/ewtn_missionary_tools.py --analytics

# Generate report
python scripts/python/ewtn_missionary_tools.py --report
```

## Routing Logic

Tickets are routed to JARVIS Superagent Team for:

- **Escalation**: Issues escalated from other teams
- **Orchestration**: System-wide orchestration needs
- **Cross-Domain**: Issues spanning multiple domains
- **Strategic**: Strategic planning and decision making
- **@jarvis**: Explicitly tagged with `@jarvis`
- **Superagent**: Superagent-level coordination needs

## Usage

### Automatic Routing
Tickets are automatically routed to JARVIS Superagent Team when they contain:
- `@jarvis` tag
- "jarvis" keyword
- "superagent" keyword
- "orchestration" keyword
- "escalation" keyword
- "cross-domain" keyword
- "strategic" keyword
- "system-wide" keyword

### Manual Assignment
```bash
python scripts/python/jarvis_c3po_ticket_assigner.py --assign HELPDESK-XXXX --team jarvis_superagent
```

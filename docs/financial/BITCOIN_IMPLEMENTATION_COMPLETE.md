### Run Example Workflows
```bash
python scripts/python/example_bitcoin_workflow.py
```

### Generate Test Data
```bash
python scripts/python/generate_bitcoin_test_data.py
```

### Generate PDFs (if tools available)
```bash
python scripts/python/pdf_generator.py
```

### Use in Code
```python
from bitcoin_jarvis_integration import BitcoinJARVISIntegration
from bitcoin_financial_workflows import ClientProfile, RiskTolerance

# Initialize
integration = BitcoinJARVISIntegration()

# Create client
profile = ClientProfile(
    client_id="CLIENT_001",
    risk_tolerance=RiskTolerance.MODERATE,
    investment_horizon_years=10,
    bitcoin_knowledge_level=4,
    financial_sophistication="accredited"
)

# Execute onboarding
success, results = integration.execute_bitcoin_onboarding_workflow(
    profile,
    portfolio_value=1000000.0,
    require_verification=True
)
```

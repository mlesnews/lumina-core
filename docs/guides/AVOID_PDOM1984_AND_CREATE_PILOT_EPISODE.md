### Production Phases

#### PHASE 1: Generate Trailer Videos (7 trailers, 30 seconds each)

**Tools**: Runway ML, Pika Labs, Stable Video

**Steps for Each Trailer:**
1. Open AI video tool (Runway ML: https://runwayml.com)
2. Input script from `lumina_pilot_trailer_videos.py`
3. Set duration: 30 seconds
4. Generate video
5. Download video file
6. Save to NAS: `Y:\LUMINA-YT\trailers\trailer_XXX.mp4`
7. Mark as produced: `python scripts/python/lumina_pilot_trailer_videos.py --produce trailer_XXX`

**Estimated Time**: 2-4 hours for all trailers

#### PHASE 2: Generate 1980s-Style Segment (15 minutes)

**Content:**
- Retro TV show format
- VHS-style graphics
- Commercial breaks
- Scan lines and retro colors

**Steps:**
1. Create 1980s-style visuals
2. Generate segments:
   - Programming intro (2 min)
   - Main content (5 min)
   - Commercial break (5 min)
   - Return to programming (3 min)
3. Apply VHS filters (scan lines, color bleed)
4. Export to NAS: `Y:\LUMINA-YT\eighties_segments\`

**Estimated Time**: 4-6 hours

#### PHASE 3: Create Main Content Segments

## 📧 Phase 2: Outreach and Contact

### Strategy 1: Craft Compelling Outreach Messages

**Email Template Structure:**

```
Subject: Partnership Opportunity: [Your Value Proposition]

Hi [Name],

[Personalized opening - reference their company/content]

I'm [Your Name] from LUMINA, [brief description of what LUMINA does].

[Why you're reaching out - specific to them]

[What you offer - value proposition]
- [Benefit 1]
- [Benefit 2]
- [Benefit 3]

[Social proof - stats, achievements]

[Call to action - next steps]

Best regards,
[Your Name]
```

**Key Elements:**
- **Personalization**: Reference their company specifically
- **Value Proposition**: What's in it for them
- **Social Proof**: Stats, achievements, case studies
- **Clear CTA**: What you want them to do next

### Strategy 2: Multi-Channel Approach

**Channels to Use:**

1. **Email** (Primary)
   - Direct, professional
   - Can include media kit
   - Easy to track

2. **LinkedIn** (Secondary)
   - Connect first
   - Build relationship
   - Then pitch

3. **Twitter/X** (Engagement)
   - Engage with their content
   - Build awareness
   - Soft pitch

4. **Direct Website Contact** (Backup)
   - Contact forms
   - Partnership inquiries
   - Media inquiries

**Best Practice:**
- Start with email
- Follow up on LinkedIn
- Engage on social media
- Use multiple touchpoints

### Strategy 3: Follow-Up Strategy

**Follow-Up Schedule:**

1. **Initial Outreach**: Day 0
2. **First Follow-Up**: Day 7
3. **Second Follow-Up**: Day 14
4. **Final Follow-Up**: Day 30

**Follow-Up Messages:**
- Reference previous message
- Add new value (new stats, content)
- Offer alternative (smaller partnership)
- Keep it brief

**Best Practice:**
- Don't give up after first email
- Provide value in each follow-up
- Respect their time
- Know when to stop

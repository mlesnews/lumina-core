## 📋 HR Processes

### Talent Acquisition Process

1. **Job Requisition** → Create job requisition
2. **Job Posting** → Post job
3. **Candidate Sourcing** → Source candidates
4. **Screening** → Screen candidates
5. **Interviewing** → Interview candidates
6. **Selection** → Select candidate
7. **Offer** → Make offer
8. **Onboarding** → Onboard new employee

### Performance Management Process

1. **Goal Setting** → Set performance goals
2. **Performance Tracking** → Track performance
3. **Performance Review** → Review performance
4. **Feedback** → Provide feedback
5. **Improvement** → Performance improvement (if needed)
6. **Recognition** → Recognize performance

### Employee Development Process

1. **Development Planning** → Create development plan
2. **Training** → Provide training
3. **Skill Development** → Develop skills
4. **Career Development** → Career planning
5. **Progress Tracking** → Track development progress
6. **Evaluation** → Evaluate development

# Cursor IDE Keyboard Shortcuts
## Quick Reference for Workflow Optimization

---

## 🎯 **WORKFLOW SHORTCUTS**

### Accept Changes:
- **Check Cursor Settings** → Keyboard Shortcuts → Search "accept"
- Common shortcuts:
  - `Ctrl+Shift+A` - Accept all (if configured)
  - `Ctrl+Enter` - Accept and continue (if configured)
  - Custom shortcut can be set in Cursor settings

### Send Message:
- **Check Cursor Settings** → Keyboard Shortcuts → Search "send"
- Common shortcuts:
  - `Ctrl+Enter` - Send message (if configured)
  - `Enter` - May send in some contexts
  - Custom shortcut can be set

### Voice Recording:
- Check Cursor Settings for voice shortcuts
- May not have pause/resume shortcuts (limitation)

---

## 📋 **HOW TO FIND SHORTCUTS**

1. Open Cursor
2. Go to **File** → **Preferences** → **Keyboard Shortcuts**
3. Search for:
   - "accept"
   - "send"
   - "apply"
   - "voice"
   - "recording"
4. Customize as needed

---

## ⚙️ **CUSTOM SHORTCUTS TO SET**

### Recommended:
1. **Accept All Changes**: `Ctrl+Shift+A` (if not exists)
2. **Send Message**: `Ctrl+Enter` (if not exists)
3. **Pause Voice**: `Ctrl+Space` (if possible)
4. **Resume Voice**: `Ctrl+Space` (toggle, if possible)

---

## 🔧 **IF SHORTCUTS DON'T EXIST**

1. Submit feature request to Cursor team
2. Use automation scripts (see `cursor_workflow_automation.py`)
3. Use voice buffer system for pause/resume
4. Use queue API for multiple submissions

---

**Check your Cursor settings first - shortcuts may already exist!**
